# $isNumber
Проверяет, является ли предоставленный текст допустимым числом.

### параметры:
| название        | тип        | описание                         | не обязательно? |
| ----------- | ----------- | ------------------------------------| -------- |
| текст        | текст      | Текст, чтобы проверить, является ли это числом.  | ложь    |

### пример:
```js
$isNumber[2.4] // правда
$isNumber[0] // правда
$isNumber[1e4] // ложь
```
