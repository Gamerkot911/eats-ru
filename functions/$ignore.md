# $ignore
просто игнорит код.

### параметры:
| название          | тип        | описание                         | не обязательно? |
| ------------- | ----------- | ----------------------------------- | -------- |
| текст          | любой         | Все, что вы хотите иметь там.    | ложь    |

### пример:
```js
$ignore[вы можете сохранять заметки или что-то еще здесь, не будучи выполненным, лол]
```
