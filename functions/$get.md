# $get
Получить значение локальной переменной.

### параметры:
| название        | тип        | описание                          | не обязательно? |
| ----------- | ----------- | ------------------------------------ | -------- |
| название        | текст      | Имя переменной для получения значения.  | ложь    |

### пример:
```js
$get[test] // ответ: ок
$var[test;ok] // необходимо использовать эту функцию
```
