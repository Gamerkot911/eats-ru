# $getQuery
Получить значение параметра запроса.

### параметры:
| название        | тип        | описание | не обязательно? |
| ----------- | ----------- | ----------- | -------- |
| название        | текст      | название параметра. | ложь |

### пример:
```js
// http://localhost:port/route?text=Hello
$getQuery[text] // Hello
```
