# $deleteVar
Удаляет переменную из базы данных.

### параметры:
```js
$deleteVar[название]
### пример:
```js
$getVar[hi] // undefined
$deleteVar[hi]
$getVar[hi] // ok
$setVar[hi;ok]
```

> ⚠: Для использования этой функции необходимо настроить базу данных. Посмотри [советы](tips.md?id=using-database)
