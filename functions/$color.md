# $color
Устанавливает цвет контекста холста.

### Parameters:
| Name        | Type        | Description                          | Optional |
| ----------- | ----------- | ------------------------------------ | -------- |
| hex        | String      | The цвет шестнадцатеричного кода для установки цвета контекста.  | false    |

### Example:
```js
$color[#303132] // Цветовой контекст будет серым.
```
