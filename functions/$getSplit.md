# $getSplit
Получите разделенный элемент из разделенного текста.

### параметры:
| название          | тип        | описание                         | не обязательно? |
| ------------- | ----------- | ----------------------------------- | -------- |
| индекс         | число      |   положение элемента.               | ложь    |

### пример:
```js
$getSplit[2] // bye
$split[hello|bye|ok;|] // необходимо использовать эту функцию
```
