# $hasText
Проверить, есть ли в коде текст.

### параметры:
| название        | тип        | описание                          |не обязательно? |
| ----------- | ----------- | ------------------------------------ | -------- |
| источник      | текст      | источник текста.                     | ложь    |
| текст        | текст      | текст для проверки.                   | ложь    |

### пример:
```js
$hasText[hello world;hello] // true
```
