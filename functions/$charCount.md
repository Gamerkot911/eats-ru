# $charCount
Получить длину текста (буквы).

### параметры:
| название        | тип        | описание                          | не обязательно? |
| ----------- | ----------- | ------------------------------------ | -------- |
| текст        | текст      | Текст для подсчета букв.   | ложь    |

### пример:
```js
$charCount[Wow] // 3
```
