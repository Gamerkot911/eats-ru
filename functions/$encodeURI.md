# $encodeURI
Кодировать текст в компоненты URI.

### Parameters:
| название        | тип        | описание                          | не обязательно? |
| ----------- | ----------- | ------------------------------------ | -------- |
| текст        | текст      | текст для кодировки.                  | ложь    |

### примеры:
```js
$encodeURI[Hello lol, hru?] // Hello%20lol%2C%20hru%3F
```
