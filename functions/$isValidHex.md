# $isValidHex
Проверяет, является ли текст допустимым шестнадцатеричным цветовым кодом.

### параметры:
| название      | тип        | описание              | не обязательно?    |
| --------- | ----------- | ------------------------ | ----------- |
| текст      | текст      | текст который проверит функция . | ложь       |

### пример:
```js
$isValidHex[#040E20] // true
$isValidHex[white] // false
 ```
